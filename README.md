# HiggsMotion — Transparent LED Film Display

Marketing website for the HiggsMotion Crystal LED Film Display product line, a brand owned by INHAUS Sdn Bhd and distributed by EMI Automation Sdn Bhd.

Live: https://higgsmotion.com

Tech stack: Next.js 15 (App Router) + TypeScript + Tailwind CSS 3 + Vercel + Cloudflare DNS
